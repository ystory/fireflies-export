# Support

## Bug reports

Use [GitHub Issues](https://github.com/ystory/fireflies-export/issues) for reproducible bugs, regressions, and documentation issues.

## Feature requests

Use [GitHub Issues](https://github.com/ystory/fireflies-export/issues) for feature requests and workflow ideas.

## Security reports

Do not open public issues for security problems.

- Prefer GitHub private vulnerability reporting when it is enabled for the repository.
- Otherwise, email [oss@ystory.kr](mailto:oss@ystory.kr) with the subject line `fireflies-export security report`.

## General questions

For installation, usage, or maintenance questions, open an issue or email [oss@ystory.kr](mailto:oss@ystory.kr).
