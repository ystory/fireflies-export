# Code of Conduct

## Our commitment

We want `fireflies-export` to be a respectful, low-friction, and welcoming open source project.

Participants in this project are expected to:

- Be respectful in technical disagreement
- Assume good intent and give actionable feedback
- Avoid harassment, discrimination, and personal attacks
- Focus on helping others learn and contribute safely

## Unacceptable behavior

This includes, but is not limited to:

- Harassing or insulting language
- Trolling, intimidation, or threats
- Publishing private information without permission
- Sexualized language or unwanted attention
- Repeated bad-faith disruption of project discussion

## Scope

This code of conduct applies to project spaces, including issues, pull requests, discussions, and maintainer communication related to the project.

## Reporting

To report unacceptable behavior, contact [oss@ystory.kr](mailto:oss@ystory.kr). Reports will be reviewed privately and handled as fairly and promptly as possible.

## Enforcement

Maintainers may edit, remove, or reject comments, issues, pull requests, or other contributions that violate this code of conduct. Repeated or severe violations may result in temporary or permanent exclusion from project spaces.
